# CONTEXTO SOULJU — Base de Conhecimento

## QUEM É A SOULJU
SoulJu é a primeira plataforma de saúde feminina por ciclos metabólicos. Não é uma marca de suplementos — é um protocolo de restauração metabólica baseado em dados reais (NHANES n=11.933), machine learning (Random Forest + K-Means), e 4 eixos metabólicos interconectados.

## OS 4 EIXOS METABÓLICOS
1. **Bioenergético Mitocondrial**: Ciclo de Krebs, CTE (Complexos I-IV + ATP sintase), β-oxidação (CPT-1/CPT-2), metabolismo NAD⁺/NADH. Produção de ~95% do ATP celular. Cofatores: CoQ10, B1/B2/B3/B5, ALA, Mg²⁺, Fe²⁺.
2. **Proteico-Estrutural**: Via mTORC1 (leucina → Rag → p70S6K), transaminação (AA + α-KG ↔ cetoácido + glutamato), sistema fosfocreatina, síntese de colágeno (prolil/lisil hidroxilase), síntese de glutationa (GCL + GSS). Massa muscular e tecido conectivo.
3. **Defesa & Reparação Celular**: Sistema glutationa (GSH/GSSG), NAD⁺/sirtuínas (SIRT1-7), autofagia (mTOR/AMPK), senescência celular (SASP), PARPs (reparo DNA). Longevidade e proteção contra ROS.
4. **Regulação Sistêmica**: Eixo intestino-fígado-hormônios, hormônios de saciedade (GLP-1, PYY, grelina), estroboloma (β-glucuronidase bacteriana), SCFAs (butirato, propionato, acetato), eixo HPA (cortisol).
