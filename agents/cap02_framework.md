## OS 4 EIXOS A DETALHAR:

### EIXO 1 — BIOENERGÉTICO MITOCONDRIAL
Centro: Glicólise → PDH → Ciclo de Krebs → CTE → ATP sintase + β-oxidação + NAD⁺
Produtos SoulJu: Fire (HALO), Morning, Cellular
Heroes: GlucoVantage® (AMPK), CoQ10 Kaneka (CTE CIII), Carnipure® (CPT-1 shuttle), PQQ (PGC-1α biogênese), ALA (PDH/α-KG DHG cofator)
Impacto hormonal: E2 → PGC-1α (biogênese), CPT-1 (flexibilidade metab.), Complexo III (leak ROS), UCP1 (termorregulação)

### EIXO 2 — PROTEICO-ESTRUTURAL
Centro: mTORC1, transaminação (PLP/B6), ciclo glutamato-glutamina, GDH → α-KG (alimenta Krebs), fosfocreatina, síntese colágeno, síntese GSH
Produtos: Sculpt, Skin, Night (Glicina)
Heroes: HMB 1.5g (↓MuRF-1 ubiquitina-proteassoma, Wilkinson 2013), Glicina 3g (NMDA + colágeno + GSH, Bannai 2012), Colágeno bioativo (Pro-Hyp/Hyp-Gly, Proksch 2014), Leucina 2.5g (threshold mTORC1)
Impacto hormonal: E2 → mTOR (resistência anabólica -30% MPS), cortisol → MuRF-1/atrogina-1, GH/IGF-1 → colágeno (-1.5%/ano pós-meno), DHEA-S ↓2%/ano

### EIXO 3 — DEFESA & REPARAÇÃO CELULAR
Centro: Sistema glutationa (GCL → γ-GluCys → GSS → GSH; GR recicla), NAD⁺/sirtuínas (SIRT1-7: deacetilação, reparo DNA, metabolismo), autofagia (AMPK ativa, mTOR inibe, ULK1 → LC3), senescência (↑p16INK4a → ↑SASP → inflamação crônica), PARPs (NAD⁺-dependentes, reparo DNA)
Produtos: Cellular, Night, Skin
Heroes: RiaGev® (+26% NAD⁺ 8 sem), Resveratrol (ativação SIRT1 via allosteric, Howitz 2003), Quercetina (senolítico: ↓SASP, Kirkland 2017), PQQ (biogênese mito, Chowanadisai 2010)
Impacto hormonal: E2 é antioxidante direto (estabiliza CIII, ↓leak ROS 2-3x), E2 modula SIRT1 via ERα, menopausa = ↑ROS + ↓GSH + ↑senescência + ↓NAD⁺ = tempestade perfeita para envelhecimento acelerado

### EIXO 4 — REGULAÇÃO SISTÊMICA
Centro: Hormônios de saciedade (GLP-1 das células L, PYY das células L, grelina das células P/D1), estroboloma (β-glucuronidase bacteriana reconjuga estrogênios → excreção biliar → reabsorção), SCFAs (butirato = combustível colonócitos + ↓inflamação; produção por fermentação de fibras), eixo HPA (CRH → ACTH → cortisol; feedback negativo comprometido no stress crônico), barreira intestinal (tight junctions: claudinas, occludina, ZO-1)
Produtos: GUT (HALO), Night (Ashwagandha)
Heroes: Glucomannan HV 3g (gel gástrico → ↑GLP-1, claim EFSA), L. gasseri cepa metabólica 3-5B (↓gordura visceral fem., Kadooka 2010), PHGG Sunfiber® 3g (Low-FODMAP prebiótico, produção SCFAs), Ashwagandha 300mg (↓cortisol 23%, Lopresti 2019)
Impacto hormonal: ↓E2 → ↓diversidade microbiota (estroboloma dysbiosis) → ↑β-glucuronidase → ↑reabsorção estrogênios tóxicos (16α-OH-E1) vs protetores (2-OH-E1), ↓progesterona → ↓GABA → ansiedade + insônia, stress crônico → ↑cortisol → ↑permeabilidade intestinal
